搭建Hexo-Github Pages博客遇到的一些问题

发表于 2018-12-01 更新于 2019-12-05
本文字数： 658 阅读时长 ≈ 1 分钟

绑定域名

首先介绍一下绑定域名的过程
1.ping github pages 的IP地址

2.把获取的IP绑定到域名上
腾讯云进入控制台，点击云解析
记录类型：A 主机记录：www 解析线路：默认 记录值：185.199.109.153
记录类型：A 主机记录：@ 解析线路：默认 记录值：185.199.109.153
3.在zhouhaojie.github.io的根目录下添加CNAME
内容为域名：grenze.cn

出错

CNAME文件在每次hexo g -d后就会消失，需要重新创建

解决

将需要上传到GitHub的内容放到source文件夹中，如CNAME
